================================
Business Requirement from Issues
================================
Introduction
============

This module is part of a set of modules 
(`Business Requirements <https://github.com/OCA/business-requirement/blob/8.0/README.md>`_) 
This module adds integration between project issues  and Business Requirement
model:

* Possibility to create a BR from a Project Issue 

Configuration
=============

Button to create the BR in the issue is available for project users.

Usage
=====

#. Link your project in the project issue.
#. Click on the Generate BR button.
#. Fill the fields in the wizard to create the BR.
